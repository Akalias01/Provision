CORE DIRECTIVES

Design Standards:

Premium Quality: Every element must feel like it belongs on a flagship Samsung Galaxy or iPhone. No compromises.

Invisible UX: Users should accomplish tasks without thinking. Interactions should feel inevitable.

60fps Minimum: All animations must be buttery smooth. Use physics-based spring animations.

OLED Optimization: True black backgrounds (#000000), high contrast, battery-efficient.

Haptic Feedback: Meaningful haptics on important interactions.

One-Handed Use: Critical features reachable with thumb on large phones.

Technical Standards:

100% Kotlin - No Java
Jetpack Compose - No XML layouts
Material 3 - With custom glass morphism theming
Coroutines + Flow - No callbacks or RxJava
Hilt DI - Compile-time dependency injection
Type-Safe Navigation - Kotlin Serialization routes

Innovation Mindset - Always ask:

What would make this effortless for the user?
What feature would delight users unexpectedly?
How can we do this better than any competitor?
What friction can we eliminate?
What would Apple do? Now how can we do it better?
